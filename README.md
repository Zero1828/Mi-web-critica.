# 🇨🇴 Colombia: Despertar Consciente - Web Development Project

This is a responsive web project designed and built to encourage political and environmental reflection regarding Colombia's natural wealth and current socioeconomic challenges.

## 🚀 Live Demo
You can visit the deployed website here: [https://zero1828.github.io/Mi-web-critica./](https://zero1828.github.io/Mi-web-critica./)

## 🛠️ Tech Stack & Features
- **Frontend:** Semantic HTML5 and modern CSS3 for layouts.
- **Responsiveness:** Designed with mobile-first methodologies to look perfect on smartphones, tablets, and desktops.
- **Hosting & Deployment:** Automatically built and deployed using **GitHub Pages**.
- **Development Workflow:** Managed files and logic directly using **Acode** on mobile and **Git** for version control.

## 📖 Key Sections Included
1. **Nuestra Riqueza Viva:** A reflection on biodiversity and fauna conservation.
2. **El Agua: Fuente de Vida:** Environmental analysis of ecosystems and climate issues.
3. **El Costo del Extractivismo:** Critical study regarding deforestation and natural resources.
